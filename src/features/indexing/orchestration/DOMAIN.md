Purpose: Define the stable boundary for indexing orchestration decomposition work. This domain is the extraction target for `runProduct` seams.

- `src/core/**`

- `src/features/<name>/index.js` only
Forbidden outbound dependencies:
- `src/features/<name>/*` internals outside `index.js` entrypoints

- retired top-level indexing folders (`src/discovery/**`, `src/extract/**`, `src/learning/**`, `src/search/**`, `src/validator/**`, `src/llm/**`)
