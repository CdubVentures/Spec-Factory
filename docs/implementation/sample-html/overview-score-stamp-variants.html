Score Grade — School Stamp Variants

Round 2: louder, more playful grade treatments. Each option still fits the existing column slot or expands it modestly. Same five grade tones as today (A green, B blue, C amber, D orange, F red).

Sample · 2026-04-25 · iteration 2
V1 · Rubber Stamp
Tilted, double-bordered, ink-bleed speckle. Square stamp with letter on top + score number under it. Each row tilts a slightly different angle so consecutive rows don't read identically.
