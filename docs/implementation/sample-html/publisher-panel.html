Implementation Architecture

Backend
API Route: src/features/publisher/api/publisherRoutes.js
GET /api/v1/publisher/:category/candidates

Query Params:
page, limit, product_id, brand, field_key, source, status, since, until, has_repairs

New Store Methods:
specDb.getFieldCandidatesPaginated(filters)
specDb.getFieldCandidatesStats(filters)

Response Shape:
{ rows: [...], total: number, page: number, stats: { total, resolved, pending, repaired, products } }
Frontend
Page: tools/gui-react/src/pages/publisher/PublisherPage.tsx

Query Hook: usePublisherCandidatesQuery(category, filters)

Table: Reuse DataTable from src/shared/ui/data-display/
with renderExpandedRow for detail panels

Nav: Register in pageRegistry.ts
Ops group, after Runtime Ops, before Review Grid

Column Defs: publisherColumns.tsx
Separate file, @tanstack/react-table ColumnDef array
