How to pick

Scalar (SKU / RDF): reply with A, B, or C. Trade-off summary:

   A — most balanced; value pill + confidence stat.
   B — value is the hero; great when the value IS the product identity (a SKU).
   C — densest; good if you often compare many rows at once.

PIF: reply with A, B, C, or D. Trade-off summary:

   A — twin bars; precise but visually heavy.
   B — spec line; text-first, zero bars.
   C — slot dots; playful, reads like a board-game progress tracker.
   D — combined bar; single visual rhythm, compact.

Cherry-picking is fine — e.g. "scalar B, PIF D, but swap the action buttons to the Option C style".
