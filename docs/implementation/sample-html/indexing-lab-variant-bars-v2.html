Variant Bar — v2 options

Scope: three brand-new scalar options (SKU / RDF) with a different aesthetic, and three PIF options that keep the slot-dots concept you liked but retire the plain "X images" text column.

Scalar (SKU / RDF) · shows: swatch · name · value · confidence · actions. No type chips.

PIF · shows: swatch · name · slot dots for views + hero · actions. Image count is derived visually from the dots or shown as a single short state word, not as a separate stat column.
