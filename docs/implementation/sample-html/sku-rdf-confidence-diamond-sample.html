Cell contract

  • Color chip: same ColorSwatch as PIF cell (18×18)
  • Diamond: polygon points=20,2 38,20 20,38 2,20 — rhombus SVG
  • Fill color: tier from confidence (0–100): ≥85 green · ≥60 amber · >0 red · =0 gray (dashed)
  • Number: Math.round(confidence) — 0 to 100, centered, white (V1) or colored (V2)
  • Label: truncated value below — SKU MPN like 910-006... or ISO date like 2024-03 (or hide and show only on hover)
  • Tooltip: {variantLabel} · {fullValue} · conf {conf}%
  • Multiple variants: wrap horizontally like PIF cell
