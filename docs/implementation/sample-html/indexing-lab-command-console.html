Indexing Lab — Command Console

A variant of the Phase 2 polish direction, pushed further into "instrument panel" territory. The top strip becomes a Command Strip with telemetry readouts (stage · variant · T+ elapsed), the finder tabs become a Console Deck attached to the panel below — each tab carries its own live mini-bar + LED + keyboard shortcut — and the active tab physically connects into the panel like a folder tab.

  • Command Strip — product chip + live telemetry slots + status LED + primary Run. Subtle grid backing for the cockpit feel.
  • Console Deck — 5 tabs (CEF · PIF · RDF · SKU · KEY). Each shows name + viewsFilled / viewsTotal mini-bar + LED + [1–5] shortcut.
  • Seamless join — active tab's bottom edge is the top of the panel. The 2 px accent band on the active tab reinforces the connection.
  • Phase chip inside the panel top bar — "STAGE 2 · PRODUCT IMAGE" so you always know where you are in the pipeline.
  • Hero KPI left, three telemetry stats right. Variant cards below with status-coloured left edge.
  • Footer status bar — one-line console bar: last run · model · pipeline health · elapsed.
