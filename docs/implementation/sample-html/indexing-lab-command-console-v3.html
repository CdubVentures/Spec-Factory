Indexing Lab — Command Console v3 (light theme, matches current GUI)

Same console density as v1/v2 but in the app's real visual language: white surfaces, soft borders, rounded corners, blue accent, professional shadows. Monospace is reserved for numbers (counts, progress, SKUs) — the rest stays in the UI sans-serif. No scanlines, no angular bevels, no corner brackets. "Command center" comes from per-tab live readouts, a telemetry strip, a phase chip inside the panel, and a bottom status bar — not from HUD chrome.

  • Command Strip — product chip, telemetry slots (Stage 2/5 · Variant · Phase · T+ · Ambig), status LED, primary Run button.
  • Console Deck — each tab: colored icon chip matching the finder's domain + name + inline mini progress bar with count + status LED + [1–5] kbd badge.
  • Active tab connects into the panel via a 2 px blue accent band on top and a shared border seam at the bottom. The panel has a blue top border and subtle lift shadow.
  • Phase chip in the panel top-bar — Stage 2 · Product Image.
  • Hero KPI (amber while incomplete, green when done) + three secondary stats.
  • Variant cards with status-coloured left edge (green/amber/red) + labeled progress bars + icon-first action buttons.
  • Status bar at the bottom — one-line summary with a 2 px blue top rule tying it to the panel.
