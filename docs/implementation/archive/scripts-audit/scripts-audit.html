Manual Scripts Audit

Spec Factory `scripts/**` Audit

Scope Discipline

This report is only for scripts/**. The inventory files only guarantee every current path is present; every status and note in this report is hand-authored after reviewing the actual script contents, outputs, and repo drift.

Method

  • Reviewed every file directly because the scripts surface is small enough to classify one by one.
  • Marked only one high-confidence dead file: the EloShapes wrapper that points at a missing module.
  • Used DEFER for scripts that still look potentially useful but currently drift through missing output roots, hardcoded run IDs, or milestone-specific assumptions.
  • Kept live browser audits, codegen, schema docs, health checks, and generic extraction utilities in the active/support buckets.
