When scoped — required work
Each Group A table needs a JSON write-back path that mirrors SQL edits to a durable JSON surface (e.g., category_authority/{cat}/_overrides/components/*.json). The rebuild seeder must then read those JSON files to restore state on a fresh DB. This brings them from rebuild partial → rebuild yes.
