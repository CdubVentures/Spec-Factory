SpecDb

Per-Category 36 tables (36 schema + 0 migration)

One SQLite database per category (.workspace/db/{category}/spec.sqlite). 36 tables from DDL (36 in specDbSchema.js + 0 migration-created). 4 additional runtime learning tables created by learningStores.js. Holds all domain data: products, components, reviews, evidence, billing, telemetry, source intelligence, URL crawl ledger, and query cooldowns.
